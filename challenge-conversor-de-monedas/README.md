# Challenge Conversor de Monedas
Archivos del challenge *Conversor de monedas* para el curso *Java Orientado a Objetos* de **Alura Latam**.

El programa realiza una petición HTTP a la API *ExchangeRate-API* para obtener datos del valor de las divisas mundiales al momento.ecibe la respuesta (cuando es exitosa) en formato JSON, que se manipula con la librería ```GSON```.

El programa permite al usuario:
- Realizar conversión entre divisas, eligiendo la divisa original y la divisa de conversión a través de su código de tres caracteres, y también eligiendo la cantidad a convertir.
- Ver los códigos de divisas disponibles (principales).
- Ver todos los códigos de divisas disponibles a través de la API.
