# Alura
Archivos relacionados con los cursos de Alura Latam / Oracle Next Education (ONE)
- [Amigo secreto](https://github.com/joseycastror/Alura/tree/main/challenge-amigo-secreto)  
  Juego que permite añadir nombres para después seleccionar uno al azar.

- [Conversor de monedas](https://github.com/joseycastror/Alura/tree/main/challenge-conversor-de-monedas)  
  Programa para obtener conversiones entre divisas a través de **ExchangeRate-API**.
  
- [LiterAlura](https://github.com/joseycastror/Alura/tree/main/challenge-LiterAlura)  
  Programa para obtener datos de libros en **Project Gutenberg** a través de la API **Gutendex**.
